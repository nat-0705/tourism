1. Open your XAMPP/WAMP's Control Panel and start the Apache and MySQL.
2. Extract the downloaded source code zip file.
3. If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory. And If you are using WAMP, paste it into the "www" directory.
4. Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
5. Create a new database naming tourism_db.
6. Import the provided SQL file. The file is known as tourism_db.sql located inside the database folder.
7. Browse the Tourism Management System in a browser. i.e. http://localhost/tourism and http://localhost/tourism/admin for the admin side.
